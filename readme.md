# A simple manga downloader for Scan-fr or frscan

A simple python script make for downloading manga on scanfr.cc or frscan.me

In order to use it just install the requirements
```
pip install -r requirements.txt
```
Then launch the script and enter the first image url (the actual url image not the page)
The url should look like this :
	```
https://www.frscan.me/uploads/manga/black-clover/chapters/0001/001.jpg
https://www.frscan.me/uploads/manga/ushijima-lusurier-de-lombre/chapters/0035/001.jpg
https://www.scan-fr.cc/uploads/manga/my-hero-academia/chapters/285/vf/001.png
```
The script will download the images in a newly creater folder called Mangas
